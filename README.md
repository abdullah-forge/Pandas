# Coffee Sales Data Analysis with Pandas

This project demonstrates basic data manipulation and analysis using the Pandas library in Python. The operations are performed on a sample coffee sales dataset in Google Colab.

## Features

- Read CSV files
- Display and inspect data
- Drop columns
- Rename columns
- Create new columns using arithmetic operations
- Apply if-else conditions
- Save selected columns to a new CSV file
- Merge multiple DataFrames
- Concatenate DataFrames
- Join DataFrames

## Files

- coffee_sales.csv
- coffee_suppliers.csv
- coffee_branches.csv
- new_orders.csv

## Technologies Used

- Python
- Pandas
- Google Colab

## Learning Objectives

This project is intended for beginners to practice common Pandas operations and understand how to manipulate and analyze tabular data efficiently.
